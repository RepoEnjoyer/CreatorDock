## 1.0.0 - 2026-07-17

### Added

- Local-first creative project briefs with priorities, target dates, tags, notes, hooks, objectives, and audiences.
- Six-stage pipeline with drag-and-drop and select-based status controls.
- Production checklists with progress tracking.
- Monthly calendar and deadline visibility.
- Five built-in workflows plus editable custom templates.
- Validated reference links and portable JSON backup import and export.
- System, light, and dark appearance modes.
- Responsive, keyboard-friendly interface and offline application shell.
- Strict backup limits, Content Security Policy, publication-hygiene checks, and automated domain tests.
